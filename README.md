# LocalConnect 🚀

## Project Overview
LocalConnect is a web-based platform that connects workers and job providers locally.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: MySQL

## How to Run
1. cd backend
2. npm install
3. node server.js
4. Open index.html
